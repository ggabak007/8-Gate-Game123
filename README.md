스테이지

- random함수로 0~100설정. 정상 < 75 < 이상으로 구현
이상현상 발생시 해결후 왔던 길로 돌아감. 이상현상 미발생시 다음스테이지로 이동

이상현상
-
이상현상은 스테이지당 1개로 제한
이상현상 발생시 도구를 습득하여 도구를 이용해 이상현상 해결
ex ) 창문에 낙서가 생김 ( 빨간색 글씨로 '이 복도를 조심해' 등으로 공포감 형성 ) , 화장실에 있는 물수건으로 닦음(상호작용 키로 일정시간 누르고있으면 해결하게) 해결 후 잠시동안 창밖에 귀신 형상이 보이게 하면 좋을듯?
ex ) 복도에 없던 그림이 생김 ( 사물함 안에 들어있는 쇠 지렛대를 이용해 그림을 제거 )
ex ) 복도의 책상에 없던 책이 생김 ( 드럼통 안에 넣고 라이터로 태움 )
ex ) 움직이는 ai (에셋스토어에서 적절한 캐릭터 다운) 
이상현상이 있으면 랜덤시간을 부여해 귀신, 인형등 공포감 조성할수있는 캐릭터 잠시동안 등장하는것도 고려
